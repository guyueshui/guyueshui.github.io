Matrix Factorization

January 3, 2019
Notes
Preliminaries Def: A matrix $A \in M_n$ is normal if $AA^∗ = A^∗A$, that is, if $A$ commutes with its conjugate transpose. Def: A complex matrix $A$ is unitary if $AA^∗ = I$ or $A^∗A = I$, and a real matrix $B$ is orthogonal if $BB^T = I$ or $B^TB = I$. There is no so-called "orthonormal" matrix. There is just an orthogonal matrix whose rows or columns are orthonormal vectors.
